# Установка и настройка Kibana

В этом разделе описана установка и базовая настройка Kibana на виртуальной машине `elk-lab`.

### Примечания перед началом

- Kibana устанавливается на ту же ВМ, что и Elasticsearch (`elk-lab`).
- Kibana подключается к Elasticsearch по HTTPS (порт `9200`).
- Веб-интерфейс Kibana будет доступен на порту `5601`.
- Для настройки подключения используется `enrollment token`

[Официальная документация Kibana](https://www.elastic.co/guide/en/kibana/current/deb.html)

## 1. Установка Kibana

Kibana устанавливается из того же репозитория, что и Elasticsearch.

```bash
sudo apt update && sudo apt install -y kibana
```

## 2. Генерация enrollment token для Kibana
Enrollment token — это одноразовый ключ, который автоматически настраивает Kibana для безопасного подключения к Elasticsearch (включая копирование сертификатов и настройку аутентификации).
```bash
sudo /usr/share/elasticsearch/bin/elasticsearch-create-enrollment-token -s kibana
```
Скопируйте полученный токен (длинная строка символов).
**Примечание**: Токен действителен 30 минут. Если срок истёк — просто сгенерируйте новый той же командой.

## 3. Настройка сетевого доступа
По умолчанию Kibana слушает только localhost. Для доступа из учебной сети нужно изменить конфигурацию.
```bash
sudo nano /etc/kibana/kibana.yml
```
Раскомментируйте и измените строку:
```bash
# Адрес, на котором Kibana будет принимать соединения
# 0.0.0.0 — слушать все интерфейсы (для доступа из сети)
server.host: 0.0.0.0
```
Остальные настройки оставьте по умолчанию — enrollment token настроит подключение к Elasticsearch автоматически.

## 4. Запуск и добавление в автозагрузку
```bash
sudo systemctl daemon-reload
sudo systemctl enable --now kibana.service
```
Проверьте статус:
```bash
sudo systemctl status kibana.service
```
**Примечание**: Первый запуск Kibana может занять 1-2 минуты. Если статус показывает activating — подождите и проверьте снова.

В выводе команды status будет показана ссылка с verification code:
```text
Kibana has not been configured.
Go to http://<host>:5601/?code=<code> to get started.
```
Запишите шестизначный verification code из этой ссылки.

## 5. Доступ к Kibana из сети
Узнайте IP-адрес ВМ elk-lab:
```bash
ip a | grep "inet " | grep -v 127.0.0.1
```
С любого компьютера в учебной сети откройте браузер и перейдите по адресу:
```text
http://IP_АДРЕС_elk-lab:5601
```
## 6. Первоначальная настройка через веб-интерфейс
При первом открытии Kibana выполните следующие шаги:  

Шаг 1: Ввод enrollment token  
    - На странице приветствия вставьте enrollment token, сгенерированный ранее.  
    - Нажмите Configure Elastic.

Шаг 2: Ввод verification code  
    - Если запрошен verification code — введите шестизначный код из вывода systemctl status kibana.  
    - Дождитесь завершения настройки (может занять до минуты).

Шаг 3: Вход в систему  
    - На странице входа введите:  
        - Username: elastic  
        - Password: пароль, сохранённый при установке Elasticsearch  
    - Нажмите Log in.

**Примечание**: Если страница не загружается сразу после запуска — подождите 1-2 минуты и обновите страницу. Kibana требуется время на инициализацию.

# Итог
- Kibana установлена из официального репозитория
- Сгенерирован enrollment token
- Настроен сетевой доступ (server.host: 0.0.0.0)
- Служба запущена и добавлена в автозагрузку
- Выполнена настройка через веб-интерфейс (enrollment token + verification code)
- Выполнен первый вход с пользователем elastic
- Веб-интерфейс доступен из учебной сети
